Chapung Art adalah galeri dan marketplace resmi komunitas seniman/perupa.

Produk utama:
- Lukisan
- Fotografi seni
- Karya digital
- Ilustrasi
- Patung
- Kriya
- Mixed media
- Art print

Model operasional:
- Hanya 5 orang internal yang memiliki akses dashboard.
- Perupa atau anggota komunitas tidak memasukkan karya secara mandiri.
- Semua karya yang dimasukkan oleh pengelola dianggap resmi dan siap dipublikasikan.
- Tidak ada fitur kurator, pilihan kurator, antrean kurasi, atau approval berlapis.
- Desain boleh terinspirasi dari struktur visual Fotografer Indonesia, tetapi tidak boleh menyalin source code, aset, logo, identitas, atau tampilan secara identik.
- Fokus utama selalu karya para perupa, profil perupa, kategori karya, dan proses penjualan.

PRINSIP TANPA BIAYA TAMBAHAN

- Pertahankan domain dan hosting yang sudah aktif.
- Pertahankan Laravel, Filament, database, Blade, Vite, Tailwind CSS, dan dependency yang sudah tersedia.
- Gunakan hanya library, font, ikon, CDN, dan layanan gratis.
- Jangan menggunakan template premium.
- Jangan menggunakan plugin berbayar.
- Jangan menggunakan font berbayar.
- Jangan menggunakan payment gateway berbayar.
- Jangan menggunakan API ongkir berbayar.
- Jangan menyarankan upgrade hosting.
- Gunakan pembayaran transfer manual dan pengelolaan ongkir manual bila sistem saat ini memakainya.
